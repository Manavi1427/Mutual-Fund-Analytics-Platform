# Bluestock Mutual Fund Analytics Dashboard

## Overview
A Mutual Fund Analytics Dashboard built using Python, SQLite and Power BI to analyze industry trends, fund performance, investor behavior and SIP market trends.

## Tech Stack
- Python
- Pandas
- SQLite
- Power BI

## Project Workflow
1. Data Ingestion
2. Data Cleaning
3. Performance Analytics
4. Advanced Analytics
5. Dashboard Development

## Dashboard Pages

### Industry Overview
- Total AUM
- SIP Inflows
- Folios
- Scheme Count

### Fund Performance
- CAGR Analysis
- Sharpe Ratio
- Alpha/Beta
- Risk vs Return

### Investor Analytics
- State-wise Investments
- Age Group Analysis
- Transaction Distribution

### SIP & Market Trends
- SIP Growth
- Category Inflows
- Benchmark Performance

## Key Metrics
- CAGR
- Sharpe Ratio
- Alpha
- Beta
- Maximum Drawdown
- Fund Score
